Formateur : Claude Couderc
Date du rip : 06/2019
Niveau : Débutant + Intermédiaire
Durée : 2h45

Description :
Dans cette formation qui s'adresse aux ingénieurs et aux administrateurs système, vous découvrirez les outils et les services d'information Internet IIS pour Windows Server 2019, le système d'exploitation pour serveurs de Microsoft. En compagnie de Claude Couderc, vous procéderez d'abord à l'installation d'IIS et à sa configuration, puis vous entrerez dans le vif du sujet avec l'administration et la gestion de pools de services et d'applications, l'optimisation des performances, la sécurité et la maintenance. À la fin de cette formation, vous aurez toutes les cartes en main pour administrer les services IIS pour Windows Server 2019.
